本文以IntelliJ IDEA 2017.3为例进行讲解。（持续更新：2018年2月2日）

适用版本（其它版本未测试）：

IntelliJ IDEA 2017.2.6

IntelliJ IDEA 2017.3

IntelliJ IDEA 2017.3.1

网络良好的情况下建议使用方法一、二；无网络情况下建议使用方法三、四。

小编建议使用方法四，不要问为什么，因为小编使用的就是这种方法。

建议下载Ultimate（For web and enterprise development）版本

方法一：使用激活服务器激活方法

由于JetBrains封杀，大部分激活服务器已经不能使用，目前可用的网址有：

http://xidea.kunx.org:5869

http://idea.java.sx

步骤

1、打开注册/激活窗口；

2、选择 License server；

3、填入上述其中一个网址；

4、点击Activate即可完美激活!

方法二：注册码激活方法

1、在hosts文件中添加如下内容：

0.0.0.0 account.jetbrains.com

注：Win XP、Win 7、Win 8、Win 10系统中，hosts文件位置C:\WINDOWS\system32\drivers\etc

2、

3、在“用户名”中输入计算机名称；

4、点击获得注册码即可获得IDEA激活码。

1、下载IDEA本地服务器

2、解压到本地硬盘任意位置，运行相应程序，32位操作系统请双击运行32位程序，64位操作系统请双击运行64位程序，如下图：

4、打开IDEA，在激活框内输入上图标示的网址即可。

注：在新版本IDEA2017.3中可能会出现无法激活的情况，此时将网址改为http://0.0.0.0:1017即可。

方法四：破解补丁激活方法

（1）破解补丁下载

将下载的破解补丁放在你的安装IDEA下面的bin的目录下面（如图），本文示例为D:\installsoft\IDEA\bin

（2）修改配置文件

在安装的IDEA下面的bin目录下面有2个文件 ： idea.exe.vmoptions和idea64.exe.vmoptions。
用记事本打开，在上述两个文件的最后一行均加上下面这段代码（红色字体部分为你的IDEA安装目录）：

-javaagent:D:\installsoft\IDEA\bin\JetbrainsCrack-2.6.10-release-enc.jar

汉化：IntelliJ IDEA 2017.3汉化

1、在主界面选择File → Settings → Appearance&Behavior → Appearance → 勾选Override default fonts by(not recommended)

Name: Microsoft YaHei（选择任意一个中文字体，否则中文显示乱码），例如小编选择Microsoft YaHei（微软雅黑）

2、然后将刚刚下载的补丁resources_cn.jar 复制到 lib 目录下，重启IDEA即可完成汉化。
